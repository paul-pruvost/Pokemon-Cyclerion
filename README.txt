# Pokémon Cyclerion

## Présentation

Pokémon Cyclerion est un fangame réalisé seul à l’aide de Pokémon Essentials v21.1. Le projet comprend :
- Le moteur RPG Maker XP intégré à Pokémon Essentials (version v21.1, intégré le 30 juillet 2023) :contentReference[oaicite:0]{index=0}  
- Des sprites et fichiers temporairement stockés dans le dossier `Sprites/Temp` :contentReference[oaicite:1]{index=1}  
- Des outils dans un répertoire `Utilities` :contentReference[oaicite:2]{index=2}
